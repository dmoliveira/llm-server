# ⚡ LLM Server

**MLX models, beautifully local.**

[GitHub](https://github.com/dmoliveira/llm-server) · [Quick start](https://github.com/dmoliveira/llm-server#-quick-start) · [Operations](https://github.com/dmoliveira/llm-server/blob/main/docs/operations.md)

LLM Server is a Mac-first local control plane for MLX-LM. It gives every model a clear alias, isolated localhost port, health state, and log trail.
